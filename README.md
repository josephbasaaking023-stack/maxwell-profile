# Maxwell Boamah Amofa - Profile Website

Personal profile website for Maxwell Boamah Amofa, advocate for human rights, transitional justice, and sustainable development in Africa.

## Features

- 📸 Interactive slideshow with photo inscriptions
- 🎓 Educational background display
- 💼 Professional experience section
- 👥 Leadership experience
- 🏆 Awards and recognition
- 🤝 Engagements with world leaders
- 🖼️ Image upload via admin panel (drag & drop)
- 💾 LocalStorage data persistence
- 📱 Fully responsive with hamburger menu
- 🎨 Wine & Gold elegant theme

## Pages

- **Home** - Slideshow and quick highlights
- **About** - Mission, vision, and core values
- **Experience** - Education and professional experience
- **Leadership** - Leadership roles and awards
- **Media** - Conferences, articles, engagements
- **Contact** - Contact information and social links
- **Admin** - Content management panel

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript
- Font Awesome Icons
- LocalStorage API
- Cloudflare Pages (hosting)

## How to Use the Admin Panel

1. Navigate to the **Admin** page
2. Upload images via drag & drop or URL
3. Edit any text content in the text areas
4. Click **"Apply Changes"** to save
5. Changes are stored in your browser's localStorage

## Deployment

This site is deployed on **Cloudflare Pages** and automatically updates when changes are pushed to GitHub.
